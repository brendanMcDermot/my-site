A Winters Knight

I created A Winters Knight as part of my Interactive Media class. This class concentrated on methodologies and process and each student was able to choose their own project. I chose to create this game as i wanted to learn more about game engines and 3D development.

loading

The Aims

The aim of this project was to build evertything myself with the exception of the game engine. Unity was selected as the engine of choice and blender was used for 3D modelling and animating.

loading

The Process

The development lifecycle envolved the use of scrum for one and the design thinking methodology. Each iteration user testing was carried out to refine the end product.

loading

Things Learnt

Through this project I learned a lot of new skills with blender, and aquired a basic introduction to unity. The design thinking methodology really helped refine the end product and learning how to animate was a slow but fun process.
