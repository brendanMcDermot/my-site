Creative numbers

I created this website in the early part of 2021 as a portfolio site to show prospective employers. The main feature in the first iteration (current iteration), was the home page. This page was supposed to catch the users attention. The particle system which was calculated in javascript and drawn on a canvas at a lower z-index seemed like a playful way to show something cool. Off course this was not particularly hard as there were many resources available for this sort of thing. So in an effort to add complexity so that the home page served to "impress". I made the values customisable and allowed them to be saved to the browsers local storage, allowing users to fully customise the home screen.

NZ Health

In the forth semester of my degree i was tasked with creating three websites. One that utilises bootstrap, a wordpress e-commerce site and a health referrals website. The task was to create this referral website with only css (no frameworks) and django. The website has full CRUD functionality, authorization and uses a Mysql database to store all the data created. The website of course also had to be fully responsive and designed with accessability in mind. Each of these websites was also running on HTTPS and was hosted on a AWS instance.
