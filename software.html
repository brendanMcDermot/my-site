File Consolidator

This project was my contribution to a colaborative effort of a group of students who took part in a new pilot class called tech colab. This class was designed to get students to create products for real world clients. The client had wanted a phone app that would work in conjunction with an RFID reader. the files created by the app would then need to be cleaned and consolidated on desktop. This was my consolidator app that i created with April Mc Mahon and Matt Osty using C#.

Python web scrapper

In the second semester at ara we were introduced to webscraping using beautiful soup. I created a python program that could scrape csv files from the lecturers website, grab the data and then clean, organise and present that data in a more human readable display.
