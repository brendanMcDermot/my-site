Thesius and the Minotaur

My Introduction to C# class got students to build a model of the game Theseus and the Minotaur and test the model. The final part of the class was to build the view in either WPF, Winform or both. I chose to do it in both. The most interesting part of this project was the automation of drawing each level. Walls would be drawn using the pen tool and arrays of vertical and horizontal co-ordinates calculated from data in the model. If you added a binary string (which was how levels where being added) then you did not have to do anything else. Everything needed by the view was already calculated.
